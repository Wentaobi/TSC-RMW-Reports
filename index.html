ROS 2 Default RMW TSC reports

Introduction

Each year, ROS 2 is released on World Turtle Day. For every ROS 2 release, the ROS 2 TSC votes on the default RMW for the release. In order for the TSC to make an informed decision, Open Robotics produces a report on the state of the available RMWs. The list below links to each of the available reports.
